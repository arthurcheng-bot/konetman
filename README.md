# 科網人 - 科技、AI最新資訊洞見

> 未來感 × 高科技 × 簡約風格的個人科技洞見站

## 🚀 快速上線（GitHub Pages）

### 步驟 1：建立 GitHub 倉庫
1. 登入 [GitHub](https://github.com)
2. 點「New repository」
3. 名稱填：`konetman`（或你喜歡的名字）
4. 勾選「Add a README file」→ Create

### 步驟 2：上傳檔案
將以下三個檔案上傳到倉庫根目錄：
- `index.html`
- `style.css`
- `README.md`（可覆蓋原本）

### 步驟 3：開啟 GitHub Pages
1. 進入倉庫 → 點「Settings」
2. 左側選「Pages」
3. Branch 選 `main` → `/ (root)` → Save
4. 等待 1 分鐘 → 刷新頁面
5. 你的網站上線！網址：`https://你的用戶名.github.io/konetman`

## 🖥️ 本地預覽
1. 下載檔案到資料夾
2. 用 VS Code 開啟
3. 安裝「Live Server」擴充
4. 右鍵 `index.html` → 「Open with Live Server」
